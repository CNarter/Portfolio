Del Dia des Muertos Memory Game

Welcome to Memory de los Muertos! A fun and challenging web-based memory game with a festive "Day of the Dead" theme. The objective is to test and improve your short-term memory by correctly repeating a sequence of flashing cards. With each successful level, you are presented with a new, more complex pattern to memorize. The game is built with vanilla HTML, CSS, and JavaScript, making it lightweight and easy to run in any modern web browser.
